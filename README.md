# 🎈 Balonel — Jocul Didiei

Joc educativ pentru copii, creat cu drag și iubire.

---

## 🚀 Deploy pe Netlify (3 pași simpli)

### Pasul 1 — Instalează dependențele
```bash
npm install
```

### Pasul 2 — Creează build-ul
```bash
npm run build
```
Se creează folderul `build/`

### Pasul 3 — Publică pe Netlify
1. Mergi la **https://netlify.com/drop**
2. Trage folderul `build/` în browser
3. Primești linkul instant! 🎉
